Neural networks are a class of models inspired by how the human brain works. They are built from small units called neurons, which are connected together in layers.

Neuron

A neuron is the basic building block of a neural network. It takes inputs, applies weights, adds a bias, and passes the result through an activation function. Mathematically: $z = \sum_{i=1}^n w_i x_i + b$

$a = \phi(z)$

where:

  • x_i = input features
  • w_i = weights (learnable parameters)
  • b = bias (learnable shift term)
  • $\phi$ = activation function (e.g., sigmoid, ReLU, tanh)
  • a = output of the neuron

So a neuron learns how important each input is (via weights) and produces a transformed output.

Perceptron

The perceptron is the simplest kind of neuron. It takes inputs, multiplies them by weights, sums them, and applies a step function: $y = \begin{cases} 1 & \text{if } \sum_i w_i x_i + b > 0 \\ 0 & \text{otherwise} \end{cases}$

Limitation: perceptrons can only solve linearly separable problems (e.g., OR, AND) but fail on XOR. This led to the development of multi layer networks with nonlinear activations.

Layers

One layer consists of multiple neurons working in parallel. Each neuron in a layer receives the same input vector, computes its own weighted sum, and outputs a value. The outputs of all neurons in a layer are collected into an output vector. If a layer has m neurons, then for an input vector $x \in \mathbb{R}^n$: $z = Wx + b$ $a = \phi(z)$ where:

  • W = weight matrix of size m \times n
  • b = bias vector of size m
  • a = output vector of size m

Neural Network (Multi layer Perceptron)

A neural network is formed by stacking layers of neurons.

  • Input layer: takes raw data (features).
  • Hidden layers: one or more layers of neurons that learn intermediate representations.
  • Output layer: produces final predictions (e.g., classification probabilities). So the structure looks like:

Input → [Hidden Layer 1] → [Hidden Layer 2] → … → [Output Layer] Mathematically, for L layers:

$a^{(0)} = x$

$z^{(l)} = W^{(l)} a^{(l-1)} + b^{(l)}$

$a^{(l)} = \phi(z^{(l)})$

for each layer l = 1, 2, …, L.

The final output $a^{(L)}$ is the prediction of the network.

Neural Networks for Regression vs Classification

Neural networks can be used for both regression and classification tasks.

The architecture (layers, neurons, activations) can look very similar.

The main difference lies in the output layer + loss function:

Regression

Output: usually a single neuron with no activation (linear output). Goal: predict a continuous value. Loss function: Mean Squared Error (MSE).

$\mathcal{L}{\text{regression}} = \frac{1}{N} \sum{i=1}^N (y_i - \hat{y}_i)^2$

Classification

Output: For binary classification: 1 neuron with sigmoid activation. For multi class classification: K neurons with softmax activation. Goal: predict probabilities for classes, assign highest probability to the true class.
Loss function: cross entropy Loss:

$\mathcal{L}{\text{classification}} = - \sum{k=1}^K y_k \log p_k$

A neural network:

working:

  1. input layers take inputs and send them to hidden layers
  2. you can define the number of layers + neurons (pytorch for example you can put as many linear layers as you want and you decide the number of neurons by deciding the number of output layers)
  3. inside each hidden layer the same wx+b processing happens where we use many layers so that the model learns as many features as possible and gives us the best prediction possible
  4. for a classification problem we might add softmaxx just before the output layer so that it gives us probabilities
  5. for a regression problem we might make it such that the output is just a single number ( if its just predicting a single number) and so on
  6. obviously before doing all of this there's a lot of preprocessing that has to take place.
