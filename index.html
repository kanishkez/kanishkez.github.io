these are some of the projects I've worked on so far:

kisangpt

it is an AI-powered agricultural assistant that supports multiple languages, image recognition and uses real-time data from reputable sources like data.gov.in to act as an assistant for Indian farmers.

link: kisangpt-idgi.onrender.com
github repo: github.com/kanishkez/KisanGPT

mcp server for obsidian

this mcp server can act as an agent for obsidian allowing claude or any other mcp host to interact with obsidian to write content and delete them in any vault.

repo: github.com/kanishkez/obsidian-mcp

hourswap

i built this as a project for my entrepreneurship class, it is basically a platform where students can leverage their skills to learn other skills or get work done by other professionals without spending a time.
it is a barter-trade system where a photographer may do some work for an editor in return for some editing done for himself. (without any money spent).

link: hourswap.vercel.app
repo: github.com/kanishkez/HourSwap
